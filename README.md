# ScholarPath

## Description

ScholarPath is my final project for Harvard CS50x.

ScholarPath is a scholarship discovery platform that helps students explore scholarships and universities across multiple countries.

It includes an AI-inspired recommendation system that recommends scholarships based on:

- Country
- Intended Major
- Academic Percentage
- IELTS Score

## Features

- User Registration & Login
- Scholarship Database
- University Database
- AI Scholarship Recommendation
- Application Tracker
- Scholarship Blog
- Responsive User Interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
